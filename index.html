A JavaScript library for building user interfaces (UI). It uses a component-based approach to create reusable and maintainable code.

NodeJS

JavaScript runtime environment that allows you to run JavaScript code outside of a web browser. This enables building server-side applications and APIs.

MongoDB

A NoSQL document database that stores data in JSON-like documents. Offers flexibility and scalability for modern web applications.
